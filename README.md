# 🌈 PhysioAI — AI-Based Physiotherapy Assistant

PhysioAI is an interactive web application that helps users perform physiotherapy exercises using **AI-powered pose detection**. It provides **real-time feedback, rep counting, and accuracy tracking** using the device camera.

---

## 🚀 Features

- 📷 AI Pose Detection using MediaPipe  
- 🔢 Automatic Rep Counting (5 reps per exercise)  
- 🎯 Real-time Accuracy Tracking  
- 💬 Interactive chat-based UI for selecting pain area  
- 🏋️ Category-based exercises (Neck, Back, Knee, Wrist, Hip)  
- 🔊 Voice feedback using Web Speech API  
- 📊 Progress report dashboard  
- 🎨 Modern responsive UI with smooth animations  

---

## 🧠 Tech Stack

- HTML, CSS, JavaScript  
- MediaPipe Pose  
- Web Speech API
